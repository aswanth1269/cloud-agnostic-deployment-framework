# Policy Driven Cloud Agnostic Deployment Framework

This project demonstrates a policy-driven deployment framework that prevents vendor lock-in by enabling containerized applications to run across multiple cloud providers.

## Features

- Docker-based containerization
- Policy-driven deployment decisions
- CI/CD automation using GitHub Actions
- Cloud-agnostic architecture

## Tech Stack

Node.js  
Docker  
GitHub Actions  
Kubernetes (planned)

## Architecture

Developer → GitHub → CI/CD → Policy Engine → Deployment Engine → Cloud
